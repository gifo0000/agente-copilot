Guía de Autenticación y Acceso Remoto

Nuevas indicaciones incorporadas de "Gestión AI - Doble factor de autenticación"

  • El sistema permite restablecer contraseñas (Ordenador, HCIS, Correo) de forma autónoma mediante SMS, Microsoft Authenticator, correo externo o clave privada.
  • Configuración: Accede al Portal Cesus > "Configurar MFA GestionAI" e identifica con DNI/NIE.
  • Métodos disponibles:
    • Microsoft Authenticator: Uso de códigos QR y notificaciones push.
    • Código SMS: Introducción de teléfono móvil para validación.
    • Correo electrónico: Obligatorio que sea una cuenta externa (no corporativa).
    • Clave privada/token: Generación de 12 posiciones para validación manual.
Soporte Técnico: Portal CESUS o vía email: inform.hupa@salud.madrid.org.

Doble Factor MFA Office 365

El MFA (Multi-Factor Authentication) añade una capa de seguridad extra combinando algo que sabes (contraseña) con algo que tienes (móvil).

Proceso de enrolamiento:

  1. Inicia sesión en www.office.com.
  2. En Información de Seguridad, selecciona "Agregar método de inicio de sesión".
  3. Sigue los pasos para vincular Microsoft Authenticator mediante el escaneo del código QR.
  4. Configura un método secundario (teléfono) para recuperación.

Guía de Instalación: VPN Sanidad (ARCO2)

Utiliza PaloAlto GlobalProtect para conectar de forma segura a los recursos de la Consejería de Sanidad.

Pasos para la conexión:

  1. Descarga el cliente desde arco2.ar.madrid.org.
  2. Instala el agente en tu equipo Windows 10 con privilegios de administrador.
  3. En el portal del agente, escribe: arco2.ar.madrid.org.
  4. Valida con tus credenciales de dominio salud.madrid.org y tu segundo factor (MFA).

Instrucciones para solicitar usuarios para HCIS

El responsable o supervisor debe enviar la solicitud a inform.hupa@salud.madrid.org incluyendo:

  1. DNI o NIE del empleado.
  2. Categoría profesional.
  3. Recomendado: DNI de un usuario de referencia con el mismo perfil de acceso.
